# CCT English 4択クイズ

CCTの英語対策用に作った、ブラウザだけで動く4択クイズアプリです。セキュリティ・IT関連の初期単語を300語収録しています。

## 主な機能

- セキュリティ・IT関連の初期単語300語

- 4択クイズのみ
- 正解すると連続正解数が +1
- 不正解なら、その単語の連続正解数が 0 に戻る
- 3回連続正解した単語は通常出題から自動で外れる
- 3連続正解数を進捗として表示
- 分野別フィルター
- 苦手単語のみ出題
- 単語追加
- JSON自動バックアップ
- JSONインポート / ダウンロード

## 使い方

1. `index.html` をブラウザで開く
2. 4択クイズで学習する
3. 進捗はブラウザのlocalStorageに保存されます

## ファイル構成

- `index.html`：画面
- `styles.css`：デザイン
- `app.js`：学習機能
